

# Лабораторная работа №2: Оценка качества работы жесткого диска

## 📚 Теоретическая справка

### CrystalDiskMark
Утилита для измерения скорости чтения и записи дисковых накопителей. Поддерживает тестирование с различными размерами блоков данных и настройкой количества циклов тестирования.

### CrystalDiskInfo
Программа для мониторинга состояния диска через технологию **S.M.A.R.T.** (Self-Monitoring, Analysis and Reporting Technology), которая отслеживает температуру, количество ошибок, время наработки и другие критические параметры.

## ❓ Контрольные вопросы

### 1. В чём разница между последовательным и случайным доступом к данным на диске?

**Последовательный доступ** — это чтение или запись данных, расположенных подряд на диске. Характерен для работы с большими файлами (видео, архивы, образы дисков). Обеспечивает максимальную скорость работы накопителя.

**Случайный доступ** — это обращение к данным, разбросанным по разным участкам диска. Характерен для работы с множеством мелких файлов (системные файлы, базы данных, программы). Скорость случайного доступа значительно ниже последовательного и является важным показателем производительности SSD.

### 2. Что означает аббревиатура S.M.A.R.T. и какие критические атрибуты она отслеживает?

**S.M.A.R.T.** (Self-Monitoring, Analysis and Reporting Technology) — технология самодиагностики, анализа и отчётности, встроенная в современные накопители (HDD и SSD).

**Критические атрибуты S.M.A.R.T.:**
- **05** — Reallocated Sectors Count (количество переназначенных секторов)
- **C5** — Current Pending Sector Count (текущее количество нестабильных секторов)
- **C6** — Uncorrectable Sector Count (количество неисправимых ошибок секторов)
- **01** — Raw Read Error Rate (частота ошибок при чтении)
- **07** — Seek Error Rate (частота ошибок позиционирования головки)
- **C4** — Reallocation Event Count (количество событий переназначения секторов)
- **03** — Spin-Up Time (время раскрутки шпинделя, для HDD)

### 3. Почему результаты тестов CrystalDiskMark могут отличаться при повторных запусках?

Факторы, влияющие на вариативность результатов:

1.  **Кеширование и SLC-кеш SSD** — после заполнения быстрого кеша скорость может снижаться
2.  **Фрагментация и расположение тестового файла** — разное физическое расположение данных на диске
3.  **Фоновые процессы системы** — работа других программ и служб Windows
4.  **Температура и троттлинг** — при перегреве контроллер SSD может снижать производительность
5.  **Режим питания и драйверы** — настройки электропитания и версии драйверов AHCI/NVMe
6.  **Размер и тип теста** — разные параметры тестирования дают разные результаты

### 4. Какие признаки указывают на скорый выход жесткого диска из строя?

**Тревожные признаки:**

1.  **Необычные звуки** (для HDD): щелчки, скрежет, повторяющиеся стуки — признак проблем с головками или шпинделем
2.  **Частые зависания системы** при обращении к файлам или открытии программ
3.  **Ошибки чтения/записи**, появление «битых» файлов, пропажа данных
4.  **Синие экраны (BSOD)** с ошибками, связанными с диском (INACCESSIBLE_BOOT_DEVICE, NTFS_FILE_SYSTEM)
5.  **Диск не определяется в BIOS** или определяется с неправильным объёмом
6.  **Критические значения атрибутов S.M.A.R.T.** (особенно 05, C5, C6)
7.  **Резкое снижение производительности** без видимых причин

### 5. Как влияет температура диска на его производительность и срок службы?

**Для HDD:**
- **Оптимальный диапазон:** 30–45°C
- **Перегрев:** повышение температуры на 5°C может сократить срок службы HDD до 2 лет
- **Низкая температура:** ниже 20°C также нежелательна (увеличивается вязкость смазки)

**Для SSD:**
- **Оптимальный диапазон:** 30–50°C
- **Высокие температуры** (выше 50-55°C):
  - Увеличивают утечку заряда в ячейках TLC/QLC
  - Снижают надёжность хранения данных при отключении питания
  - Могут вызывать троттлинг (снижение производительности для охлаждения)
  - Ускоряют деградацию ячеек памяти
- **Контроль температуры** особенно важен при интенсивной записи и в жарких условиях эксплуатации

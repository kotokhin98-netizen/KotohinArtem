## 📦 О программе

**Passmark PerformanceTest** — профессиональный бенчмарк-инструмент для комплексной оценки производительности компьютера.

### 🔹 Ключевые задачи утилиты:
```
✓ Проверка соответствия реальной производительности заявленным спецификациям
✓ Сравнение с эталонными системами и конфигурациями конкурентов
✓ Объективная аналитика при выборе комплектующих
✓ Возможность создания кастомных сценариев нагрузочного тестирования
```

### 🔹 Группы стандартных тестов:

| Группа | Что оценивается |
|--------|----------------|
| **CPU** | Математические вычисления, шифрование, сжатие, работа с инструкциями SSE/AVX |
| **2D Graphics** | Отрисовка интерфейса, работа со шрифтами, битовые операции |
| **3D Graphics** | Рендеринг сцены в DirectX 8.1/9, анимация, шейдеры |
| **Memory** | Скорость чтения/записи, латентность, эффективность кэширования |
| **Disk** | Последовательное и случайное чтение/запись, время доступа |
| **Optical** | Скорость работы CD/DVD-приводов |

## ❓ Ответы на контрольные вопросы (уникальные формулировки)

### 1. Для каких целей может быть использована протестированная система?

Исходя из полученных метрик (PassMark 1103, CPU Mark 4158, 3D Mark 161), данная конфигурация пригодна исключительно для:

```
📄 Минимально необходимых офисных операций:
   • Набор текста в Word / LibreOffice Writer
   • Просмотр PDF-документов
   • Работа с электронными таблицами малого размера (<100 строк)

🌐 Базовый веб-серфинг:
   • Одновременная работа с 2–3 вкладками браузера
   • Просмотр видео в разрешении 720p (без fullscreen)
   • Использование мессенджеров (Telegram, WhatsApp Web)

🎓 Образовательные нужды начального уровня:
   • Прохождение онлайн-курсов без интерактивных элементов
   • Скачивание учебных материалов
   • Работа с презентациями (просмотр, не создание сложных анимаций)
```

> ❗ Система **не способна** корректно работать с современными ОС (Windows 11, macOS Sonoma) без серьёзных лагов. Рекомендуется использовать лёгкие дистрибутивы Linux (Lubuntu, Xubuntu) или Windows 10 LTSC.

### 2. Какие параметры требуют немедленного улучшения?

| Компонент | Проблема | Решение | Эффект |
|-----------|----------|---------|--------|
| **ОЗУ** | Всего 4.5 ГБ доступно | Добавить ещё 4–8 ГБ (итого 8–12 ГБ) | Устранение "подкачки", плавная работа браузера |
| **Накопитель** | Disk Mark 1427 — низкий | Замена на NVMe SSD (даже бюджетный) | Загрузка ОС за 10–15 секунд, быстрый отклик программ |
| **Процессор** | CPU Mark 4158 — устаревший | Замена на современный 4–6 ядерный (Ryzen 5 5500 / i3-12100) | Прирост производительности в 2–3 раза |
| **Графика** | 3D Mark 161 — непригодна для игр | Установка дешёвой дискретной карты (GT 1030 / RX 6400) | Возможность запуска старых игр, ускорение видеоплеера |

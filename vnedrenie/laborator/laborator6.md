## 📚 Теоретическая справка

### Инструментарий для диагностики GPU

Для выполнения работы использовались две специализированные утилиты:

#### 🔹 GPU-Z
Лёгкая утилита для детального анализа параметров видеокарты. Поддерживает адаптеры NVIDIA и AMD/Intel.

Возможности:
- Отображение модели GPU, техпроцесса, версии BIOS
- Информация о частотах в режимах 2D/3D и при разгоне
- Данные о видеопамяти: тип, объём, разрядность шины
- Сведения о поддержке API (DirectX, OpenGL, Vulkan)
- Мониторинг температур и загрузки в реальном времени

#### 🔹 FurMark / FluidMark
Стресс-тесты на базе движка OpenGL для проверки стабильности видеосистемы под экстремальной нагрузкой.

Назначение:
- Выявление артефактов и перегрева
- Оценка максимальной производительности (FPS)
- Проверка эффективности системы охлаждения
- Тестирование стабильности при длительной нагрузке

## 📊 Результаты стресс-тестирования

### 1️⃣ FurMark: Тест с эффектом "бублика" (Torus rendering)

Нагрузка на GPU с отрисовкой сложной 3D-сцены.

| Метрика | Значение (FPS) |
|---------|---------------|
| Минимум | 3 |
| Среднее | 3 |
| Максимум | 5 |

Наблюдения:
- Низкий FPS обусловлен высокой сложностью сцены и ограничениями интегрированной графики
- Артефакты изображения не зафиксированы
- Стабильность кадров: ✅ без резких просадок

---

### 2️⃣ FurMark: Тест без эффекта "бублика" (упрощённая сцена)

| Метрика | Значение (FPS) |
|---------|---------------|
| Минимум | 50 |
| Среднее | 90 |
| Максимум | 130 |

Анализ:
- При снижении графической нагрузки производительность возрастает ~4.5×
- Система демонстрирует предсказуемое поведение под нагрузкой
- Перегрев или троттлинг не обнаружены

---

### 3️⃣ FluidMark: Тест на базе физики частиц (PhysX)

Проверка производительности при расчёте физических эффектов.

| Параметр | Значение |
|----------|----------|
| FPS (Min) | 3 |
| FPS (Avg) | 4 |
| FPS (Max) | 10 |
| Температура GPU | 76 °C |

Оценка температурного режима:
- ✅ 76°C — допустимое значение для длительной нагрузки
- ⚠️ Рекомендуется обеспечить хорошую вентиляцию корпуса при продолжительных нагрузках
- 🔧 Критический порог для Intel HD: ~100°C (сработает троттлинг)

## 📝 Аналитический вывод

### Оценка производительности:
1.  Интегрированная графика Intel HD показала ожидаемые результаты для своего класса:
    - Подходит для офисных задач, веб-сёрфинга, воспроизведения видео до 1080p
    - Не предназначена для современных 3D-игр или ресурсоёмких графических приложений

2.  Стабильность работы:
    - За время тестирования (более 20 минут суммарно) сбоев, артефактов или вылетов не зафиксировано
    - Температурный режим в пределах нормы, троттлинг не активировался


3.  Влияние нагрузки:
    - Разница между тестами с разной сложностью сцены подтверждает корректную работу динамического масштабирования частот GPU

### Рекомендации:
- ✅ Для повседневных задач текущей конфигурации достаточно
- ⚠️ Для работы с 3D-графикой или играми рекомендуется дискретная видеокарта
- 💡 Регулярно очищать систему охлаждения от пыли для поддержания температурного режима

## ❓ Контрольные вопросы

<details>
<summary><b>1. Какие программы используют для тестирования видеокарт?</b></summary>

Бенчмарки и стресс-тесты:
- 3DMark — комплексное тестирование игровой производительности
- Unigine Heaven / Superposition — кроссплатформенные бенчмарки на движке Unigine
- PassMark PerformanceTest — оценка GPU в составе общей системы
- MSI Kombustor — стресс-тест на базе FurMark с дополнительными функциями
- Heaven Benchmark — проверка стабильности разгона

Мониторинг и диагностика:
- GPU-Z — информация о параметрах видеокарты
- HWMonitor / HWiNFO — контроль температур и напряжений
- MSI Afterburner — разгон и мониторинг в реальном времени
</details>

<details>
<summary><b>2. Какие программы используют для тестирования мониторов?</b></summary>

Диагностика дисплеев:
- PassMark MonitorTest — комплексная проверка цвета, контраста, геометрии
- Dead Pixel Tester — поиск «битых» пикселей
- ScreenTest — онлайн-инструмент для быстрой проверки
- EIZO Monitor Test — профессиональная калибровка и тестирование
- UXPilot — набор тестов для веб-разработчиков и дизайнеров
</details>
